About GoMatha Application.


This application is developed in interest to store data of cows in Shri Krishna Ashramam Pedamuthevi.

Description of Application


There are three segments in this Application.

Store Data


This Store data segment stores the data of the cow like Location,Name,Number Given,Breed,DOB,Gender,Health Condition,Parents Diseases,Medicines Used,Other Information.
Please note that this Store data is to input data for new cow only(First Time Entry).

Modify/Delete Data


This Modify/Delete Data segments delete's the cow whole entry.Or modifies entries of cow.
As this is first version of his application please Note that all entries are to be copy paste fields like medicines used and other information first and proceed update.
Here id is to be entered first so that the entered ID details will be updated.Update will be done by ID basis.

View Data


Enter the filter detail you want to search and select appropriate button to get result
